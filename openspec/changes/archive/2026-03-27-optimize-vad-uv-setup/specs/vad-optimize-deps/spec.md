## ADDED Requirements

### Requirement: Python dependencies declared in pyproject.toml
`optimize/VAD/pyproject.toml` SHALL declare all runtime dependencies (`pyminidsp`, `optuna`, `soundfile`, `numpy`, `scipy`) and require Python >= 3.10. It SHALL NOT include a `[build-system]` section.

#### Scenario: pyproject.toml contains all required dependencies
- **WHEN** a developer reads `optimize/VAD/pyproject.toml`
- **THEN** the `[project].dependencies` array MUST list `pyminidsp`, `optuna`, `soundfile`, `numpy`, and `scipy`

#### Scenario: Python version constraint is specified
- **WHEN** a developer reads `optimize/VAD/pyproject.toml`
- **THEN** `requires-python` MUST be set to `>=3.10`

### Requirement: Lockfile committed for reproducible installs
`optimize/VAD/uv.lock` SHALL be generated by `uv lock` and committed to version control, pinning exact dependency versions.

#### Scenario: Lockfile generated successfully
- **WHEN** a developer runs `uv lock` in `optimize/VAD/`
- **THEN** a `uv.lock` file MUST be created that pins all transitive dependencies

#### Scenario: Lockfile is committed
- **WHEN** the change is complete
- **THEN** `optimize/VAD/uv.lock` MUST be tracked by git

### Requirement: Virtual environment artifacts excluded from version control
The repo-root `.gitignore` SHALL include `.venv/` and `__pycache__/` so auto-created virtual environments and bytecode caches are never committed.

#### Scenario: .venv ignored globally
- **WHEN** uv creates a `.venv/` directory anywhere in the repo
- **THEN** `git status` MUST NOT show it as an untracked file

#### Scenario: __pycache__ ignored globally
- **WHEN** Python creates `__pycache__/` directories anywhere in the repo
- **THEN** `git status` MUST NOT show them as untracked files

### Requirement: README documents uv-based workflow
`optimize/VAD/README.md` SHALL document how to run the optimization script using `uv run` and note that uv must be installed as a prerequisite.

#### Scenario: README shows uv run usage
- **WHEN** a developer reads `optimize/VAD/README.md`
- **THEN** it MUST include `uv run python optimize_vad.py` as the invocation command

#### Scenario: README mentions uv prerequisite
- **WHEN** a developer reads `optimize/VAD/README.md`
- **THEN** it MUST mention that uv needs to be installed, with a pointer to installation instructions

### Requirement: Script executes successfully via uv run
Running `uv run python optimize_vad.py --help` from `optimize/VAD/` SHALL resolve all dependencies and display the script's help text without errors.

#### Scenario: Dependency resolution and script execution
- **WHEN** a developer runs `uv run python optimize_vad.py --help` from `optimize/VAD/`
- **THEN** uv MUST install all dependencies into a virtual environment and the script MUST print its help output without import errors
