# Future Feature Ideas

## Spectrogram-to-Audio Reconstruction
Reconstruct audio from a magnitude-only spectrogram image using inverse STFT and Griffin-Lim phase reconstruction. See `dev_docs/spectrogram_reconstruction_plan.md` for full design.

## Morse Code Generator/Decoder
Generate Morse code audio from text and decode Morse audio back to text. Similar flavor to the existing DTMF feature.
