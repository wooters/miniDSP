## Task 2: Implement feature extraction helpers

**File**: `src/minidsp_vad.c`

Add static helper functions:

- `compute_spectral_entropy(psd, num_bins)` — normalize PSD to a probability distribution, return negative sum of p*log(p), normalized to [0, 1] by dividing by log(num_bins).
- `compute_spectral_flatness(psd, num_bins)` — geometric mean / arithmetic mean of PSD bins. Returns 0.0–1.0 (1.0 = white noise, 0.0 = pure tone).
- `compute_band_energy_ratio(psd, num_bins, sample_rate, N, band_low_hz, band_high_hz)` — sum PSD bins in band / total PSD sum.

No public API for these — they are internal to the VAD module.

**Tests**: Indirectly tested through Task 5 tests (features_out verification).

## Task 5: Implement MD_vad_process_frame

**File**: `src/minidsp_vad.c`

Processing pipeline:
1. Compute five raw features (same as calibrate).
2. Update adaptive normalization.
3. Normalize features to [0.0, 1.0].
4. Compute weighted score: `sum(weights[i] * norm[i])`.
5. Apply state machine:
   - Score >= threshold: increment onset_counter; if >= onset_frames, set decision=1, reset hangover_counter to hangover_frames.
   - Score < threshold and decision==1: decrement hangover_counter; if == 0, set decision=0, reset onset_counter.
   - Score < threshold and decision==0: reset onset_counter.
6. Write score_out and features_out if non-NULL.
7. Increment frames_processed, return current_decision.

**Tests**:
- `test_vad_silence` — process silence frames, verify decision stays 0 and score is low.
- `test_vad_speech` — process sine wave frames in speech band, verify decision becomes 1 after onset period.
- `test_vad_hangover` — speech frames followed by silence, verify decision holds for hangover_frames then drops.
- `test_vad_onset` — fewer than onset_frames of speech, verify no false trigger.
- `test_vad_features_out` — verify features array is populated, all values in [0.0, 1.0].
- `test_vad_custom_weights` — set one weight to 1.0, rest to 0.0, verify score tracks that single feature.
